### Mutate

An imbedded music client of sorts for 4chan's music board

Finds all YouTube links and queries them into a list that can be played at will

### Hotkeys:

| Key    | Effect              |
| ------ | ------------------- |
| n      | Next track          |
| p      | Previous track      |
| enter  | Load selected track |
| space  | Pause / Play        |
